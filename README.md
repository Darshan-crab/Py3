

ASSIGNMENT OVERVIEW
----------------
This assignment consists of two Python scripts designed to 
demonstrate functional programming and the use of the 
built-in 'math' library.

-----------------------------------------------------------
TASK 1: FACTORIAL USING A FUNCTION
-----------------------------------------------------------
Description:
A custom-built function named 'factorial(n)' that computes 
the product of all positive integers up to n.

Key Features:
- Handles negative input (returns an error message).
- Handles base cases (0 and 1).
- Uses an iterative loop for calculation.

Mathematical Logic: 
n! = n * (n-1) * (n-2) * ... * 1

-----------------------------------------------------------
TASK 2: USING THE MATH MODULE
-----------------------------------------------------------
Description:
An interactive script that accepts a numerical input from 
the user and performs three distinct mathematical 
calculations using Python's 'math' module.

Calculations Included:
1. Square Root: Calculates the principal square root.
2. Natural Log: Calculates the log base 'e'.
3. Sine: Calculates the sine of the value (assumes radians).

-----------------------------------------------------------
HOW TO USE
-----------------------------------------------------------
1. Requirements: 
   - Python 3.x installed.

2. Running the scripts:
   - Open your terminal or command prompt.
   - Navigate to the folder containing the files.
   - Run using the command: python <filename>.py

3. Expected Output:
   - Task 1 will print the factorial of 5 (120).
   - Task 2 will prompt you to type a number and then 
     display the results of the three calculations.

-----------------------------------------------------------
AUTHOR NOTES
-----------------------------------------------------------
Ensure that for Task 2, you enter a positive number to 
avoid math domain errors when calculating the square root 
and natural log.
